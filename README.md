 Gym API

  A production-ready REST API built with Java 17 + Spring Boot for tracking gym workouts, scheduling training weeks, and
   managing exercises — secured with JWT and deployed via Docker on Railway.

  Features

  - JWT Authentication — Stateless, secure login and registration flow
  - Workout Logging — Create sessions and track exercises with sets, reps, and weight
  - Exercise Library — Browse, filter by muscle group, and create custom exercises
  - Weekly Scheduling — Plan training days and get smart suggestions for unworked muscle groups
  - Swagger UI — Fully documented and interactable API
